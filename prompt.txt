PROJECT NAME: LocalSync Drive

Build a complete full-stack web application that allows browsing, uploading, downloading, deleting, renaming, and managing files stored on an Android device exposed through an FTP server (CX File Explorer FTP Server).

TECH STACK:

* Backend: Python FastAPI
* Frontend: HTML, CSS, JavaScript
* UI Framework: Bootstrap 5
* Icons: Bootstrap Icons
* Template Engine: Jinja2
* File Transfers: Python ftplib
* Optional: WebSocket support for live updates
* Responsive design for desktop and mobile

FTP CONFIGURATION:

* Host: configurable
* Port: configurable
* Username: configurable
* Password: configurable
* Store credentials in environment variables

GOAL:
Create a browser-based file manager similar to Google Drive, Windows Explorer, or File Browser.

20. Important Requirement

The FTP backend must work with CX File Explorer FTP server on Android.

The application must browse Android folders such as:

* Download
* DCIM
* Pictures
* Movies
* Music
* Documents

without mounting the phone filesystem locally.

Use direct FTP communication through Python ftplib.

Provide production-ready code with comments and modular architecture.
